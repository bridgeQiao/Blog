  1. 在win下安装cygwin

    日 14 一月 2018

    1. 从官网下载setup_x86_64.exe，注意这是64位的，大小在1M左右，安装到硬盘，选择网络安装，镜像网站可以选用USTC或者163；

    2. 下载apt-cyg，将该文件放到安装目录下的bin目录；

    3. bash # 换源 apt-cyg update -m http://mirrors.ustc.edu.cn/cygwin/x86_64/

    4. 现在可以用apt-cyg安装软件了，使用方式类似apt

    5. 可以安装常用的软件：gcc/g++/make

  2. 在ubuntu16.04下安装Matlab2017a

    五 29 九月 2017

    1. 从szu内部ftp下在MATLAB；

    2. 安装matlab，首先给MATLAB的install和 bin/glnxa64 加权限

    chmod -R 777 install
chmod -R 777 bin/glnxa64

    1. 需要jre环境，如果没配置，需要先安装 openjdk-8-jdk

    2. 运行install开始安装，如果没出现安装界面，选择指定jre目录, -javadir /usr/lib/jvm/java-8-openjdk-amd64/jre/

    3. 用硬件opengl, 打开MATLAB后，在控制台里输入

    opengl('save','hardware')

  4. 安裝manjaro

    六 08 四月 2017

    • 烧入usb，由于电脑是bios的，将u盘的EFI文件夹重命名为back

    • 由于我用的版本是mate live，图形界面安装有点问题，在安装软件包的时候看不到进度，所以我下载了manjaro-architect，用命令行安装：sudo setup，不得不说live文件驱动很好，wifi能正常连接

    • 用命令行安装的时候，先分好区，然后挂载，如果安装过archlinux的话，应该知道这点，接着设置软件源，然后安装基础软件、图形界面之类的，装完重启

    • 先不登录（lightdm起不来），alt+ctrl+F2，F2到F6都行，su后设置密码，添加用户

    • 进去后设置语言：manjaro-settings-manager

    • 安装常用软件：yaourt -S jdk8-openjdk android-studio typora，主要是用来学习android

    • 开机后使用pacman时，经常会出现错误，要使用sudo pacman -Syy，先刷新软件源
