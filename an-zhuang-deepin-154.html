安装deepin 15.4

二 07 三月 2017

  1. dd到一个U盘上（直接用deepin-installer.exe也可以，但用这个在15.4上只能装个启动iso，还得装在硬盘上，而15.3则不会）；

  2. 设置bios，用u盘启动，直接安装；

  3. 在桌面上右键，设置热区设置，将几个角的热区取消，触发比较麻烦，还不如直接点击；

  1. 安装字体fonts-inconsolata,冬青黑体,infinality
deb http://ppa.launchpad.net/no1wantdthisname/ppa/ubuntu xenial main

其中xenial是ubuntu的版本号，还得用 apt-key adv --keyserver keyserver.ubuntu.com --recv-keys E985B27B，添加key

  1. 搭建Android开发环境：下载android-studio和android_SDK,复制到opt下，再解压缩， 安装openjdk1.8： sudo apt install openjdk-1.8-jdk ,运行 /opt/android-studio/bin/studio.sh ，android-studio启动后，在弹出sdk界面上选择取消，在config下的sdk设置里将下好的sdk路径填进去，在新建一个项目时会下载gradle3.3-all.zip文件，由于下载速度很慢，我是先安装mini迅雷： sudo apt install apps.com.xunlei.mini. ，当然也有个qqdownload，下好后直接替换就可以了 编写android-studio.desktop:放入/usr/share/applications/
[Desktop Entry]
Name=Android Studio
Comment=Develop android program
Exec=/opt/android-studio/bin/studio.sh
Icon=/opt/android-studio/bin/studio.png
Terminal=false
Type=Application
Categories=Development;IDE;
  1. 安装redshift、vscode sudo apt install redshift，设置 redshift-gtk -l 39.92:116.46 -t 5500:4500, 在 /etc/apt/sources.list.d/ 下新建vscode.list文件:
deb [arch=amd64] https://packages.microsoft.com/repos/vscode stable main

导入GPG：

curl https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > microsoft.gpg
sudo mv microsoft.gpg /etc/apt/trusted.gpg.d/microsoft.gpg
